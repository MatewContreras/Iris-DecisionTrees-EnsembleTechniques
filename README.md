## 🧩 What is Iris-DecisionTrees-EnsembleTechniques?

This application helps you understand classic machine learning using the Iris dataset. It builds decision trees based on two popular methods: entropy and Gini. You can also see the ID3 algorithm at work, created from scratch in the code. The app includes ensemble learning techniques such as Random Forest and Bagging. These methods show how combining multiple models can improve the accuracy and reliability of predictions.

You don’t need any technical or programming skills to use this app. It’s made to run easily on Windows computers. The goal is to give you hands-on experience with basic machine learning concepts through a simple interface.

## 🔧 Troubleshooting

- **App won’t start:** Make sure you have downloaded the `.exe` file fully. Try running as Administrator by right-clicking the app and choosing “Run as administrator.”

- **No data loads:** Confirm the `Data` folder is in the same location as the executable and the `iris.csv` file is inside it.

- **Graphs not saving:** Check if the `Graphs` folder exists. If not, create a new folder named `Graphs` beside the `.exe` file.

- **Windows warning about app:** This is normal for new apps. Click “More info” and then “Run anyway” if safe.

- **Slow or no output:** Your PC may be processing. Wait a few moments or close other programs to free memory.

## 🧑‍💻 About the Iris Dataset

The Iris dataset is a simple, well-known collection of measurements from three types of iris flowers. It’s used here to teach how machines can learn to classify data. Each row shows measurements like petal length and width. The goal is to predict the flower type based on these details.

## 🛠 How It Works Behind the Scenes

This app uses Python scripts wrapped in a simple Windows program. It loads the Iris data, processes it with machine learning libraries, and applies custom algorithms. It generates charts to show training progress and comparison of results. Models are trained using well-known methods from the scikit-learn library and the new ID3 algorithm coded from scratch.

Even if you do not interact with the code, you get to see and understand the output it produces.
